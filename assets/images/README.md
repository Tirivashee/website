## Image Specifications

- **Format**: JPG or PNG
- **Resolution**: Minimum 1200px width for collections, 800px for products
- **Aspect Ratio**: 
  - Collections: 16:9 or 4:3
  - Products: 1:1 (square)
  - Team: 3:4 (portrait)
- **File Size**: Optimize to under 500KB per image
- **Color**: Use high-quality images with good contrast
